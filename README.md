# Analytica IDA

Sistema de gerenciamento multi-tenant com controle de acesso rigoroso e auditoria.

## Como Iniciar

1. **Banco de Dados (Porta 5434):**
   ```bash
   docker compose up -d
   ```

2. **Backend (Porta 5100):**
   ```bash
   cd backend
   dotnet run
   ```

3. **Frontend (Porta 3000):**
   ```bash
   cd frontend
   npm run dev
   ```
